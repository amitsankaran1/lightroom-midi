|-----------|-------|-------------|

| `Shadows2012` | -100 to 100 | Darkest areas |

## Getting Parameter Information Programmatically

You can also query Lightroom for parameter information:

```javascript
// Get all parameter names
const params = await lrClient.getParameterNames();

// Get a parameter's range
const [min, max, precision, smallInc, largeInc] = await lrClient.getRange('Exposure2012');

// Get the user-visible label
const label = await lrClient.getLabel('Exposure2012');

// Get the default value
const defaultValue = await lrClient.getDefault('Exposure2012');
```

## Notes

- Parameters ending in `2012` are from Lightroom's Process Version 2012 and later
- Some parameters may not be available in all versions of Lightroom
- Precision values indicate decimal places (0 = integer, 1 = one decimal, etc.)
- Boolean parameters use 0 (false) or 1 (true)
